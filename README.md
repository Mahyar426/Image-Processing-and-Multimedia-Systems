This repository contains laboratory exercises, homework solutions, and projects from the Multimedia Systems course, emphasizing digital image processing, watermarking techniques, and signal analysis using MATLAB.

Repository Contents

Homework 1

Basic image processing operations

Mean Square Error (MSE) computations and image analysis

Homework 2

Histogram equalization (global and local)

Image enhancement techniques and histogram analysis

Homework 3

Discrete Cosine Transform (DCT) implementation and analysis

Median filtering for noise removal in images

Homework 4

Image fusion and hybrid image creation using frequency domain processing

Hybrid image analysis combining high-pass and low-pass filtering

Project

Implementation of image watermark embedding and extraction

Robustness analysis against various attack methods

Normalized correlation calculations and watermarking security evaluation

Learning Objectives

Develop understanding and practical skills in multimedia processing techniques

Gain experience in image processing, frequency domain analysis, and watermarking

Implement key multimedia processing algorithms

Analyze and improve image quality using advanced signal processing techniques

Tools and Software

MATLAB for algorithm implementation, image processing, and simulation
